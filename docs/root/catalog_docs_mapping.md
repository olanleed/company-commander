## 3. 更新手順

### 3.1 新規車両追加時

1. `data/catalog/vehicles_XXX.json` に車両エントリ追加
2. 本対応表の該当国セクションに Vehicle ID 行を追加
3. 対応 Detail Doc（`XXX_army_weapons_2026.md`）に武装情報追加
4. `weapon_data.gd` に新規武器が必要な場合は追加

### 3.2 既存車両の武装変更時

1. `data/catalog` の該当車両を修正
2. Detail Doc の該当セクションを更新
3. 変更理由・出典を Detail Doc に記載
4. 対応テストを更新（`tests/test_*_weapons.gd`）

### 3.3 整合性チェック

将来的に以下を自動化:
- 全 vehicle ID が本対応表に存在するか
- 全 weapon ID が Detail Doc に記載されているか
- リンク切れがないか
