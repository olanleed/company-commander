# Company Commander - 操作方法

## マウス操作

### 選択
| 操作 | 動作 |
|------|------|
| 左クリック | ユニット選択（他の選択を解除） |
| Shift + 左クリック | ユニットを選択に追加 |
| 左ドラッグ | ボックス選択（範囲内のユニットを選択） |
| Shift + 左ドラッグ | ボックス選択（既存の選択に追加） |

### コマンド
| 操作 | 動作 |
|------|------|
| 右クリック | スマートコマンド（状況に応じて自動判定） |
| 右クリック長押し (0.2秒) | Pie Menu（放射状コマンドメニュー）を表示 |

### スマートコマンド判定
- **地面** → 移動
- **敵ユニット** → 攻撃
- **味方/中立の拠点** → 防御
- **敵拠点** → 攻撃

### 修飾キー（コマンド実行時）
| キー | 効果 |
|------|------|
| Alt + 右クリック | 道路優先で移動 |
| Shift | キュー追加（予定） |
| Ctrl | 強制実行（予定） |

## キーボード操作

### コマンドホットキー
| キー | コマンド |
|------|----------|
| Q | Move（移動） |
| W | Defend（防御） |
| E | Attack（攻撃） |
| R | Recon（偵察） |
| A | Suppress（制圧射撃） |
| D | Smoke（煙幕） |
| F | Support（支援） |
| X | Break Contact（離脱） |

### ゲーム速度
| キー | 速度 |
|------|------|
| 1 | 一時停止 |
| 2 | 1倍速 |
| 3 | 2倍速 |
| 4 | 4倍速 |
| Space | 選択ユニットにカメラを向ける |

### カメラ操作
| キー | 動作 |
|------|------|
| 矢印キー / WASD | カメラ移動 |
| PageUp | ズームイン |
| PageDown | ズームアウト |

### その他
| キー | 動作 |
|------|------|
| Escape | 選択解除、コマンドキャンセル |
| Enter | 全味方ユニットを選択（テスト用） |

## Pie Menu（放射状メニュー）

右クリックを0.2秒以上長押しすると表示されます。

```
        Move (N)
           |
  Suppress-+-Defend
      /    |    \
 Support   +   Attack
      \    |    /
    Smoke--+--Recon
           |
        Break (S)
```

- マウスを方向に動かしてコマンドを選択
- 右クリックを離すと実行
- 中央でキャンセル
- 左クリックでキャンセル

## HUD構成

### 上部パネル
- **TICKETS**: チケット残量
- **RP**: 増援ポイント
- **OBJECTIVES**: 拠点保持状況
- **TIME**: 経過時間
- **SPEED**: ゲーム速度

### 左パネル（部隊一覧）
- フィルタボタン:
  - **All**: 全ユニット
  - **!**: 未命令ユニット
  - **S**: 制圧状態ユニット
  - **A**: 弾薬不足ユニット
- クリックで選択

### 右パネル（選択詳細）
- **STRENGTH**: 兵力（緑バー）
- **SUPPRESSION**: 制圧度（黄バー）
- **CURRENT ORDER**: 現在の命令
- **STATE**: ユニット状態
- **POSITION**: 現在位置

### 下部コマンドバー
- 8つのコマンドボタン
- ホットキー表示付き

### ミニマップ（右下）
- マップ全体の縮小表示
- ユニット位置表示
- カメラ範囲表示（白枠）
- クリックでカメラジャンプ

### アラートログ（ミニマップ上）
- 戦闘イベント通知
- 接触報告、被射撃、損耗など
- クリックで該当位置にジャンプ
